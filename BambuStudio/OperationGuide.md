## 操作指南

| 输入         | 效果   | 说明           |
| ---------- | ---- | ------------ |
| ⌘ + 双指平移   | 平移画面 | 平移摄像机        |
| 双指移动       | 缩放画面 | 摄像机焦距变大或变小   |
| 单指按压平移     | 旋转画面 | 以屏幕中心为原点旋转画面 |
| ⌘ + 单指按压平移 | 框选   | -            |
